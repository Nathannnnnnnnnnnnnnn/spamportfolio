Writing Samples

Wiki Netplay Guide

The official wiki's page on how to play Pokemon: Close Combat online, rewritten by me. Improvements include:

  • Added reference images to help users follow step-by-step instructions
  • Reorganized information into topics and added table of contents to help users navigate the page
  • Documented several tools available in the Lilithport application to help new users learn how to use it
  • Added text markups to help users see what they're supposed to click
  • Simplified language to help users understand instructions
  • Rewrote all text to follow Microsoft's Writing Style Guide

MLA Guide

A guide for teaching middle and high school students how to format their essays for MLA. Document includes:

  • Simple, concise language to help users new to English or computers follow along
  • Google Docs screenshots with highlighted sections to show students where on their screen to click
  • Model MLA formatting for students to compare their own progress

AI Logic Documentation

Revised documentation on the AI's decision making process in Pokemon Run & Bun. Improvements include:

  • Despite adding content, shortened the document by over 1000 words by rewriting information concisely
  • Added tables to help users visualize AI decisions more clearly
  • Categorized all information and added table of contents to help users navigate the page
  • Added embedded hyperlinks throughout the document to help users quickly jump to relevant sections
  • Standarized inconsistent language written by many contributors so that each concept is always referred to by the same name
